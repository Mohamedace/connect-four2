# Connect Four Game

## Overview
This is a classic **Connect Four** game implemented using HTML, CSS, and JavaScript. Two players alternate turns, dropping pieces into a grid. The first player to align four pieces in a row, column, or diagonal wins the game. The game also highlights the winner's pieces in yellow and announces the winner's name dynamically.

## Features
- Two-player mode with customizable names.
- Dynamic winner announcement using the recorded names.
- Highlighting the winning pieces in yellow.
- Animated background and user-friendly interface.
- Responsive design for optimal gameplay experience.

## Technologies Used
- **HTML5**: For structuring the game interface.
- **CSS3**: For styling the game, including animations and responsiveness.
- **JavaScript (ES6)**: For game logic and dynamic DOM manipulation.

## How to Play
1. Open the `index.html` file in your web browser.
2. Enter the names for **Player 1** and **Player 2** when prompted.
3. Players take turns clicking on the column where they want to drop their piece.
4. The first player to align four pieces horizontally, vertically, or diagonally wins.
5. The game highlights the winning pieces and displays the winner's name.

## File Structure
- `index.html`: The main HTML file for the game interface.
- `connect_four_project.css`: Stylesheet for the game's design and animations.
- `connect_four_project.js`: JavaScript file containing the game logic.

## Setup Instructions
1. Download or clone the project to your local machine.
2. Ensure all files (`index.html`, `connect_four_project.css`, and `connect_four_project.js`) are in the same directory.
3. Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox).

## Game Details
### Dynamic Features
- The players' names are entered at the start and displayed throughout the game.
- The winner's pieces are highlighted in yellow.

### Styling Highlights
- Animated background using `fezbg.mp4`.
- Smooth transitions and hover effects.

### Logic Highlights
- The game board dynamically updates after each move.
- Automatic detection of winning combinations (horizontal, vertical, diagonal).
